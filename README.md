# loopback-react-redux-boilerplate

A deadly simple boilerplate for Loopback + React + Redux.

## Including Lib

* Loopback @3.x
* React @15.x
* Redux @3.7.x
* React Router @4.x
* Recompose
* Restful API with Redux
* Material-UI @next
* Babel
* Webpack

## Inclusive functions

* Login and register flow
